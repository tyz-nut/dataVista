**dataVista（数据视界）** 是一个面向零基础开发者的数据可视化大屏项目，旨在帮助大家从 0 到 1 学习如何亲手制作一个数据可视化大屏。

## ✨ 项目特色

- 🎯 **从零开始** — 每一步都有详细说明，适合完全没有数据大屏开发经验的初学者
- 📦 **循序渐进** — 从最基础的环境搭建，到复杂的图表交互，逐步深入
- 🔧 **实战导向** — 不做玩具项目，构建真正可以部署使用的数据大屏
- 📖 **文档完备** — 每个阶段都有对应的教程文档，跟着做就能学会
